How to name a function

Functions usually are verbs, as they're doing something.

  • get... (returns a vlue)
  • show... (calculates something)
  • create... (creates something)
  • check... (checks something and returns a boolean,etc.)

Examples: showMessage, getAge, calcSum, createForm, checkPermission
Should only be used for it's name and action and not another action

Two types of error messages
  • Reference error - the variable hasn't been defined yet
  • Trace Error - traces back to the origin of when it was called
Name 3 types of errors
  • Syntax Error - Grammatical rule, forgetting a comma or paranthesis (on console) it will show "Uncaught SyntaxErorr"
  • Reference Error - Check for spelling or check if the variable has been declared or not
  • TypeError - Trying to use a function for a variable that isn't allowed to do. Ex. using push into a string

Synchronus vs Asynchronus Code

Synchornous code flow is from top to bottom, Example: 1-10

The way it executes is from top to bottom so one must go before the other. But sometimes if one code from top is taking a long time to execute and the code on bottom doesn't need the top code to function, then this creates a slow work flow.

Asynchronus code flow solves this. Asynchronus code allows one set of functions to go while another set is running also
